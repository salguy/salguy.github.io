# 🖥 프론트엔드 개발

React 기반의 자막을 표기하는 스피커의 UI입니다.

## 주요 기술 스택
- React + Vite + TailwindCSS
- TypeScript
- Material UI
- Axios (REST API 연동)

## 페이지 구성
- 홈 화면 (자막)
- 알림 설정/관리 페이지(개발예정)

## 실행 방법
```bash
cd frontend
npm install
npm run dev
